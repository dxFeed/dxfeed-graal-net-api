WithSources(params IndexedEventSource[])

Sets the sources from which to subscribe for indexed events. If no sources have been set, subscriptions will default to all possible sources.

The default value for this source is an empty set, which means that this model subscribes to all available sources. These sources can be changed later, after the model has been created, by calling SetSources(params IndexedEventSource[]).
